这是一份有关嵌入式的综合项目，
以RTC时钟为基准，附带了ADC、DAC模块的使用，
同时分别配置了WIFI的AP的TCP服务器模式和WIFI的STA的TCP客户端模式来对时间进行校准，
以及还可以玩俄罗斯方块这个小游戏。

This is a comprehensive embedded project that takes the Real-Time Clock (RTC) as the reference. 
It incorporates the utilization of Analog-to-Digital Converter (ADC) and Digital-to-Analog Converter (DAC) modules. 
Meanwhile, it configures the TCP server mode for WIFI Access Point (AP) and the TCP client mode for WIFI Station (STA) respectively to perform time calibration. 
In addition, it also supports playing the Tetris mini-game.
